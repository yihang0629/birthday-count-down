# 智能生日倒计时器

一个基于 Vue 3 开发的智能生日倒计时器，支持公历和农历生日的管理和倒计时计算。

## 功能特性

- 🎉 支持公历和农历生日的添加和管理
- ⏰ 实时倒计时显示，精确到秒
- 📅 支持直接输入公历日期或农历日期
- 🔄 自动计算下一个生日的日期
- 📱 响应式设计，适配不同屏幕尺寸
- 💾 本地存储，数据持久化
- 🌟 生日当天自动提醒
- 📋 按距离生日的时间排序显示

## 技术栈

- Vue 3 + Composition API
- Vite
- Lunar JavaScript (农历计算库)

## 安装与运行

### 1. 克隆项目

```bash
git clone <repository-url>
cd birthday-count-down
```

### 2. 安装依赖

```bash
npm install
```

### 3. 开发环境运行

```bash
npm run dev
```

### 4. 构建生产版本

```bash
npm run build
```

## 使用方法

1. **添加生日**：
   - 输入姓名
   - 选择输入方式（公历或农历）
   - 输入出生日期
   - 选择以后过哪种生日（公历或农历）
   - 点击"保存"按钮

2. **查看倒计时**：
   - 在生日列表中点击"查看倒计时"按钮
   - 页面顶部会显示该生日的实时倒计时

3. **删除生日**：
   - 在生日列表中点击"删除"按钮
   - 确认删除操作

## 数据存储

所有生日数据都存储在浏览器的本地存储（localStorage）中，数据键名为 `birthdays_v3`。

## 核心功能说明

### 农历计算

使用 Lunar JavaScript 库进行农历和公历之间的转换，确保农历生日的准确计算。

### 倒计时算法

- **公历生日**：直接计算当前日期到下一个公历生日的时间差
- **农历生日**：先将农历生日转换为当年的公历日期，再计算时间差

### 排序算法

根据距离下一个生日的天数进行排序，距离越近的生日排在越前面。
